# 五子棋/Gobang
这是一个基于Vue + TypeScript开发的五子棋小游戏，也是我用TypeScript开发的第一个程序。

## 项目环境安装/Project setup
```
yarn install
```

### 编译和热更新(开发环境)/Compiles and hot-reloads for development
```
yarn run serve
```

### 编译和压缩(生产环境)/Compiles and minifies for production
```
yarn run build
```
